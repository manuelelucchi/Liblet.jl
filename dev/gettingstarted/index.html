Getting Started

Installation

You can install Liblet.jl throught Pkg

For example: 

julia> import Pkg

julia> Pkg.add("Liblet")
   Updating registry at `~/.julia/registries/General`
   Updating git-repo `https://github.com/JuliaRegistries/General.git`
[?25l[2K[?25h  Resolving package versions...
ERROR: expected package `Liblet [5ecd16ac]` to be registered

or 

(1.4) pkg> add Liblet

Once you have it installed, you can

julia> import Liblet

or

julia> using Liblet

Enjoy!
