# [Getting Started](@id gettingstarted)

## Installation

You can install Liblet.jl throught [Pkg](https://docs.julialang.org/en/v1/stdlib/Pkg/index.html)

For example: 

```@repl
import Pkg
Pkg.add("Liblet")
```

or 

```julia
(1.4) pkg> add Liblet
```

Once you have it installed, you can

```@repl
import Liblet
```

or

```@repl
using Liblet
```

Enjoy!
